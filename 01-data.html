Feature Engineering

After merging all sources on county FIPS, 37 structural features were retained. Several features required non-trivial engineering choices:

BEA Data Suppression Floors

The Bureau of Economic Analysis suppresses small cell values to protect privacy. Three energy-related features are heavily affected:

Suppression Floor % at Floor Implication
Feature
epg_natural_gas 5 76.5% Mostly uninformative — reflects suppression, n...
clean_energy_jobs 15 47.9% Nearly half of counties are at the floor
grid_infrastructure_jobs 20 15.9% Moderate suppression — still useful for rankings

Suppression indicator flags (*_above_floor) were added as binary features to allow the model to distinguish true low values from suppressed ones. This means SHAP values for these features should be interpreted carefully — a negative SHAP is often driven by the county being at the suppression floor rather than genuinely having low activity.

Electricity Price Aggregation

EIA electricity prices are reported at the utility zip code level. The pipeline aggregates to county level by:

  1. Mapping zip codes to counties via the ZCTA crosswalk
  2. Computing a population-weighted average for counties with multiple utility territories
  3. Separately retaining commercial (commercial_price) and industrial (industrial_price) rates

Policy Direction Score

State-level data center policy stances were scraped from legislative databases and classified on a scale of −1 (hostile) to +1 (supportive) using GPT-4 assistance, then validated manually. This feature is assigned uniformly to all counties within a state.

Feature Correlation Structure

Pearson correlation matrix for a subset of key structural features.

Notable correlations:

  • clean_energy_jobs and grid_infrastructure_jobs are strongly correlated (r ≈ 0.75) — both capture power sector depth and will compete for SHAP attribution.
  • wage_information correlates moderately with num_datacenters (r ≈ 0.45) — the agglomeration signal visible even in raw correlations.
  • Electricity prices (commercial_price, industrial_price) show weak raw correlation with DC counts, but emerge as important at scale in the Stage 2 model.

Preprocessing Pipeline

# Simplified view of the preprocessing steps
df = merge_all_sources(on="county_fips")   # left-join across 7 sources
df = drop_counties_with_missing_core(df)   # requires grid + land price data
df = add_suppression_flags(df)             # *_above_floor binary indicators
df = log1p_transform(df, skewed_cols)      # right-skewed count features
df = standardize(df, price_cols)           # z-score for price features
df = fill_missing(df, strategy="median")   # remaining NaN → column median
